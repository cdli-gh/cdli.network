Frequently Asked Questions

About CDLI Network

The cdli network is a hub for cdli mirrors. There are mirrors available on https://cdli.network/ and some are under preparation . If you and your institution would be interested in hosting a cdli mirror, do not hesitate to contact us.

What is a mirror?

A server mirror is a copy of a server which give access to the same services as the master instance from which it was copied. Some mirrors can be exact replica, others can offer an older copy that is periodically updated. Read the description under the different mirror on the main page for more information.

Why this hub?

THE CDLI Network Website is a hub that displays status and links to the CDLI mirrors around the world. It is not necessary to use this site to access your usual CDLI mirror but it might come in handy when there is a network outage in your region. In the description of each mirror, you can see the current status of that instance so when you think there is an outage, you can verify the information here.

Advantages of mirrors

• Load distribution

• Redundancy in case one of the servers is down

• Less latancy using a server close to home

• Empowers local group over their own access to open data

• Perservation & sustainability
